PRICE CLASS UNIT TESTING

List of files:
    priceClass: Contains Price.cc, Price.h, test.cc, Makefile, and various price object files, some containing errors.
    priceTest.py: Testing script for Price class.
    results.txt: Contains results of unit testing in textfile format.

Compliling and Executing:
    1. In current directory, use the command 'python3 priceTest.py' to test the program.
    
NOTE: Use the command 'make clean' inside priceClass directory to delete all test executables.
